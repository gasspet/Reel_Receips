# Reel Receips

Reel Receips ist ein statischer HTML/CSS-Prototyp fuer eine Rezept-Web-App.
Die Website zeigt eine Startseite, eine Rezeptdetailseite und eine Seite zum
Hinzufuegen neuer Rezepte.

## Projektziel

Das Projekt sammelt Social-Media-Rezepte an einem Ort, damit sie spaeter
schneller wiedergefunden und organisiert werden koennen.

## Enthaltene Seiten

- `index.html` - Startseite mit Navigation, Suche, Filtern und Rezeptkarten
- `rezept.html` - Detailansicht eines einzelnen Rezepts
- `hinzufuegen.html` - Formularansicht fuer neue Rezepte
- `imprint.html` - einfaches Impressum

## Lokal oeffnen

1. Repository herunterladen oder klonen
2. Die Datei `index.html` im Browser oeffnen

Es ist fuer den aktuellen Stand kein Build-Schritt und kein Server noetig.

Der aktuelle Milestone ist ein statischer Prototyp mit HTML und CSS.
